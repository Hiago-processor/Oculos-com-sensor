# Como rodar o código
1. Instalar o programa gratuito: Baixe a Arduino IDE em → arduino.cc/en/software

2. Conectar o Arduino no PC Plugue o cabo USB no Arduino e no computador. O PC vai reconhecer automaticamente.

3. Colar o código: Abra a Arduino IDE, crie um novo projeto e cole o código que eu escrevi.

4. Enviar para o Arduino: Clique no botão "→ Carregar" (Upload). Em segundos o código está gravado.

5. Desconectar e usar: Depois de gravado, pode desconectar o USB e ligar a bateria 9V. O Arduino roda o código sozinho, sem precisar de PC.
